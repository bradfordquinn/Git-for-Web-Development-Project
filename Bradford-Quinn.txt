    -1. What is Semantic HTML? 
     HTML that gives meaning to our content.

    -2. What is HTML used for?
     To provide information to a reader/browser to interpret so page elements and deisgn to be displayed as desired 

    -3. What is an attribute and where do we put it? 
     A modifier that controls an elements behavior, i.e. 'src'

    -4. What is the h1 tag used for? How many times should I use it on a page?
     defines the most important heading

    -5. Name two tags that have required attributes
     <img> <a>

    -6. What do we put in the head of our HTML document?
     meta data for the reader/browser to preload items such as CSS etc. 

    -7. What is an id? 
     unique id for html elements, i.e. id ='about', for easy reference in style sheets, etc.

    8. What elements can I add an id to?
     any element, but it can only be used once per element

    -9. How many times can I use the same id on a page?
     once, they are unique

    -10. What is a class?
     name(s) for an element in html, points to a style sheet
     in other words, a box placed around an element to easily define 
     segregated operations around the contents

    -11. What elements can I add a class to? 
     any element

    -12. How many times can I use the same class on a page? 
     as many times as needed

    -13. How do I get my link to open in a new tab?
     <a href="url" target="_blank">text</a> - apply target
     attribute to anchor

    -14. What is the alt attribute in the image tag used for?
     alt is used to describe an image - i.e, 'a big green tractor'

    -15. How do I reference an id?
     call in CSS: #idName{}, in HTML: #idName - requires #

    -16. What is the difference between a section and a div
     section is a specific group of elements that goes together
     div is a generic container of variable size and use

    -17. What is CSS used for?
     CSS is used to create style sheets and to develop elements 
    -18. How to we select an element? Example - every h2 on the page
     create a selector for that element, i.e. p{}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    ID is a specific reference to an individual element.
    Class is broad, and can be applied to many elements.

    Use an ID to define a section of a page to easily anchor. <section id="about"> <a href="#about">
    Use a class to define a set of parameters to be applied to multiple sections. <section class="bordered"> 
    (gave example of .bordered{} below for another answer)

    -20. How do we select classes in CSS?
     class selectors reference to class name and have a '.' (i.e. .bordered{})

    -21. How do we select a p element with a single class of “human””?
     p.human{}
    
    -22. What is a parent child selector? When would this be useful? 
     When nesting elements the 'parent' is higher in heirarchy, and child lower.
     Selecting elements such as all paragraphs within a div, div > p {}, can apply 
     styles to those elements quickly
   
    -23. How do you select all links within a div with the class of sidebar?
     div > a.sidebar{}
  
    -24. What is a pseudo selector?
     used to define a special state of an element, i.e. ':hover'

    -26. What do we use to change the spacing between letters?

        border-style: dotted;
